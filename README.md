# SS BUILDERS MVS

Welcome to the official website repository for **SS BUILDERS MVS**, a premium construction and building services company based in Melvisharam, Tamil Nadu.

## 🚀 Live Demo
**[Visit the Live Website](https://ss-builders-mvs.vercel.app)**

## 🌟 Overview
SS BUILDERS MVS is dedicated to delivering high-quality residential and commercial construction projects. This website showcases our portfolio, services, materials, and the transparent process we follow to build dreams into reality. 

The website features a modern, cinematic design with smooth scroll animations, immersive 3D elements, and a responsive layout tailored for all devices.

## 🛠️ Technologies Used
- **Frontend Core:** HTML5, CSS3, Vanilla JavaScript
- **Animations:** [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/) with ScrollTrigger for high-performance scroll animations
- **Responsive Design:** CSS Grid, Flexbox, and Fluid Typography clamps
- **3D Graphics (optional modules):** WebGL/Canvas for immersive visual sections
- **Hosting / Deployment:** [Vercel](https://vercel.com/)

## 📂 Project Structure
- `index.html` - The cinematic home page and entry point
- `gallery.html` - A comprehensive portfolio of past and ongoing projects
- `process.html` - Detailed step-by-step construction methodology
- `materials.html` - Information on premium materials and quality assurance
- `/css` - Modular stylesheets (`style.css`, `hero-cinematic.css`, `animations.css`, `pages.css`, `sections-v2.css`)
- `/js` - JavaScript logic, GSAP orchestration (`main.js`, `animations.js`, `components.js`, and 3D modules)
- `/images`, `/assets` - Compressed visual assets and media

## 📱 Features
- **Cinematic Hero Sections:** Engaging first impressions with dynamic scrolling effects.
- **Interactive Lightbox Gallery:** Seamlessly view project images in high resolution on both desktop and mobile.
- **Fluid Responsiveness:** Adapts beautifully from ultra-wide monitors down to narrow mobile screens without layout breaks.
- **Performance Optimized:** Carefully managed scroll observers and guard clauses to ensure animations only run when in view and relevant.

## 💻 Local Development
To run this project locally, you don't need any complex build tools. You can serve the files directly using any local web server.

1. Clone the repository:
   ```bash
   git clone https://github.com/ansuu27tech/ssbuildersmvs.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ssbuildersmvs
   ```
3. Use a local server to run the project. For example, if you have Python installed:
   ```bash
   python -m http.server 8000
   ```
   Or using Node.js (`serve`):
   ```bash
   npx serve .
   ```
4. Open your browser and go to `http://localhost:8000` or the provided local URL.

## 🚀 Deployment
This project is configured for seamless deployment on **Vercel**. 
Any pushes to the `main` branch will automatically trigger a production deployment.

To deploy manually via Vercel CLI:
```bash
npx vercel --prod
```
